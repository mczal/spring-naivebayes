Error Rate

True Positive True Negative False Positive False Negative
* tp = True Positive
* tn = True Negative
* fp = False Positive
* fn = False Negative

Accuracy

$$Accuracy = {tp + tn \over tp+tn+fn+fp}.$$

Recall

$$R = {tp \over tp + fn}.$$

Precision

$$P = {tp \over tp + fp}.$$

F Measure

$$ \alpha = {coefficient} $$
$$ \alpha = {2PR \over P + R} .$$
$$F = {1 \over {\alpha{1 \over P}+(1-\alpha) {1 \over R} }}.$$
